## 实现计划：教师管理系统

**技术方案：**
- 纯Java实现，控制台交互
- 分层架构：Entity → DAO → Service → Controller → Main
- 使用ArrayList模拟数据存储

**目录结构：**
```
src/main/java/com/teacher/
├── entity/Teacher.java          # 教师实体类
├── dao/TeacherDao.java          # 数据访问接口
├── dao/impl/TeacherDaoImpl.java # 数据访问实现
├── service/TeacherService.java  # 业务逻辑接口
├── service/impl/TeacherServiceImpl.java # 业务逻辑实现
├── controller/TeacherController.java    # 控制器
└── Main.java                    # 程序入口
```

**实现步骤：**

### 步骤1：创建Teacher实体类
- 属性：id, name, age, subject, phone, email
- 包含getter/setter和toString方法

### 步骤2：创建DAO层
- TeacherDao接口：定义CRUD方法
- TeacherDaoImpl：使用ArrayList实现数据存储

### 步骤3：创建Service层
- TeacherService接口：业务方法定义
- TeacherServiceImpl：调用DAO实现业务逻辑

### 步骤4：创建Controller层
- 处理用户输入，调用Service

### 步骤5：创建Main入口
- 控制台菜单交互界面

**功能清单：**
- 添加教师
- 删除教师
- 修改教师信息
- 查询单个教师
- 查询所有教师
- 退出系统
