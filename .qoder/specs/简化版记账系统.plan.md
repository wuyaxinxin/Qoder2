## 简化版记账系统

**技术方案：** 简化三层架构 + 内存存储

**创建文件（共4个）：**

| 文件 | 说明 |
|------|------|
| `entity/Record.java` | 记录实体（类型、金额、备注、日期） |
| `dao/RecordDao.java` | 数据操作（增删查） |
| `service/AccountService.java` | 业务逻辑（统计余额） |
| `Main.java` | 控制台菜单入口 |

**核心功能：**
- 添加收入/支出
- 查看所有记录
- 查看余额统计
- 退出

**验证：** javac编译 + java运行测试
