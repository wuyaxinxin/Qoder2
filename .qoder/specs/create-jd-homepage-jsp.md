# 京东首页风格 JSP 页面实现规范

## Overview

创建一个独立的 JSP 页面（`jd_index.jsp`），仿照京东首页的布局和风格，包含顶部导航栏、轮播图、商品分类侧边栏和商品推荐列表。

## Scope

**待创建文件：**
- `/Users/admin/Documents/testQoder/Qoder2/jd_index.jsp` - 主页面（内联CSS/JS）

**不涉及：**
- 不修改现有Flask项目文件
- 不创建额外的CSS/JS外部文件

## Design Approach

采用单一JSP文件方案，将HTML、CSS、JavaScript全部内联，便于独立部署到Tomcat环境。使用京东经典红色主题（#e1251b），三栏式布局设计。

## Implementation Plan

### 步骤1: 创建JSP文件基础结构
- 添加文件头注释：`始终生效`、`自由万岁`
- JSP页面指令 `<%@ page ... %>`
- HTML5文档结构

### 步骤2: 编写内联CSS样式
- 全局重置样式
- 主题色变量（--jd-red: #e1251b）
- 页面布局容器（1200px居中）

### 步骤3: 实现顶部导航栏

### 步骤4: 实现轮播图+侧边栏区域

### 步骤6: 编写JavaScript交互
- 轮播图自动播放（3秒间隔）
- 指示器点击切换
- 左右箭头导航
- 鼠标悬停暂停

### 步骤7: 添加响应式样式
- ≥1200px: 5列商品
- 992-1199px: 4列
- 768-991px: 3列，隐藏侧边栏

## Considerations

- 图片使用 `via.placeholder.com` 占位图服务，无网络时可能显示空白
- 使用ES5语法确保浏览器兼容性
- 商品数据为模拟数据，实际应用需对接后端接口
