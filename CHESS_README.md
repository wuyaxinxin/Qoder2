# 简单象棋游戏

## 项目说明

这是一个基于Java的简化版中国象棋游戏,采用控制台交互方式,实现了基本的象棋规则和双人对弈功能。

## 文件列表

- `Side.java` - 阵营枚举类(红方/黑方)
- `PieceType.java` - 棋子类型枚举类
- `Piece.java` - 棋子类
- `Board.java` - 棋盘管理器类
- `RuleValidator.java` - 规则验证器类
- `ChessGame.java` - 游戏控制器类(包含main方法)

## 编译方法

```bash
javac Side.java PieceType.java Piece.java Board.java RuleValidator.java ChessGame.java
```

或者简单地:

```bash
javac *.java
```

## 运行方法

```bash
java ChessGame
```

## 游戏规则

1. **输入格式**: `x1,y1 x2,y2`
   - 例如: `4,9 4,8` 表示将(4,9)位置的棋子移动到(4,8)

2. **坐标系统**:
   - 横坐标x: 0-8 (从左到右)
   - 纵坐标y: 0-9 (从上到下,黑方在上端y=0,红方在下端y=9)

3. **棋子移动规则**:
   - **将/帅**: 只能在九宫格内移动,每次一步,横竖方向
   - **士**: 只能在九宫格内沿斜线移动,每次一步
   - **象/相**: 沿田字对角线移动,不能过河,移动路径中心不能有棋子
   - **车**: 横竖直线移动,无距离限制,路径上不能有其他棋子
   - **马**: 走日字,移动方向的直线上不能有棋子(蹩马腿)
   - **炮**: 横竖直线移动,吃子时必须隔一个棋子
   - **兵/卒**: 未过河只能向前,过河后可左右前,每次只能走一步,不能后退

4. **胜负判定**:
   - 吃掉对方将帅即获胜

## 技术特点

- **纯Java实现**: 无需额外依赖,只需JDK即可运行
- **控制台交互**: 简单直观,易于理解
- **完整规则验证**: 实现了各种棋子的移动规则
- **详细注释**: 每个类和方法都有详细的JavaDoc注释
