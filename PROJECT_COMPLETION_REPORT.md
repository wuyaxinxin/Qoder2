## 📝 业务规则总结

### 课程管理规则
1. 课程ID必须唯一,格式为2-3个大写字母+3位数字(如CS101)
2. 学分范围: 0.5-10.0
3. 课程容量范围: 1-500人
4. 课程删除时会自动处理已选课学生的退课

### 选课规则
1. 学生不能重复选择同一门课程
2. 课程满员时不能选课
3. 退课后自动释放课程容量
4. 退课时成绩记录会被清除

### 成绩管理规则
1. 成绩范围: 0-100分
2. 只能为已选课的学生录入成绩
3. 成绩同步更新到学生对象和选课记录
4. 支持成绩修改

### 统计分析规则
- 选课率 = 已选人数 / 课程容量 × 100%
- 及格率 = 及格人数 / 总人数 × 100% (60分及格)
- 优秀率 = 优秀人数 / 总人数 × 100% (85分优秀)
- GPA计算: 加权平均 = Σ(成绩 × 学分) / 总学分
